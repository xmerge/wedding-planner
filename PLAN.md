# 结婚良辰吉日计算器 - 技术方案

## 一、项目概述

**目标：** 纯前端应用，根据新人双方出生日期时间计算八字，推算结婚良辰吉日。

**技术栈：** React + TypeScript，纯前端无后端依赖。

---

## 二、八字排盘原理

### 2.1 八字结构

```
八字 = 四柱（年柱、月柱、日柱、时柱）
每柱 = 天干 + 地支（共8个字）

天干（10个）：甲 乙 丙 丁 戊 己 庚 辛 壬 癸
地支（12个）：子 丑 寅 卯 辰 巳 午 未 申 酉 戌 亥
```

### 2.2 年柱计算（以立春为界）

```
关键：年柱以立春交接时刻为准，非春节

年干 = (年份 - 3) % 10 → 天干索引
年支 = (年份 - 3) % 12 → 地支索引
```

### 2.3 月柱计算（按节气）

```
关键：八字月份按节气划分，非农历月份

节气月份：
立春 → 正月（寅月）开始
惊蛰 → 二月（卯月）开始
...以此类推

月干公式（五虎遁）：
甲/己年：寅月起丙寅
乙/庚年：寅月起戊寅
丙/辛年：寅月起庚寅
丁/壬年：寅月起壬寅
戊/癸年：寅月起甲寅
```

### 2.4 日柱计算

使用 lunar-javascript 库处理，避免公式边界问题。

### 2.5 时柱计算（五鼠遁）

```
时干起法：
甲/己日 → 甲子时起
乙/庚日 → 丙子时起
丙/辛日 → 戊子时起
丁/壬日 → 庚子时起
戊/癸日 → 壬子时起

地支时辰：
子时 23:00-01:00  丑时 01:00-03:00
寅时 03:00-05:00  卯时 05:00-07:00
辰时 07:00-09:00  巳时 09:00-11:00
午时 11:00-13:00  未时 13:00-15:00
申时 15:00-17:00  酉时 17:00-19:00
戌时 19:00-21:00  亥时 21:00-23:00
```

## 三、择日规则

### 3.1 硬红线（一票否决）

- 岁破日：日支冲年支
- 月破日：日支冲月支
- 四离日：立春/立夏/立秋/立冬前一天
- 四绝日：春分/秋分/夏至/冬至前一天
- 候选日冲新人日支
- 新人生辰日
- 杨公忌（13个固定农历日）

### 3.2 加分项（100分制）

| 项目 | 权重 |
|------|------|
| 黄历宜嫁娶 | 30 |
| 八字匹配（三合/六合） | 30 |
| 避冲负分 | 20 |
| 周末/节假日 | 15 |
| 双数日期 | 5 |
